# Music_Extended
Antes de executar o código, faça:
 * pip3 install pydub
 * pip3 install tkinter, caso necessário
  
  
Observações:
 * Arquivos de músicas devem ser previamente baixados no formato mp3
 * arquivo Mp3 e o main.py devem estar na mesma pasta
